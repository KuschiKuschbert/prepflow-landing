## To Remap Names

If the names don't match the avatars correctly, update `lib/avatar-names.ts` in the `CUSTOM_AVATAR_NAMES` object.

## Available Name Categories

- **Chef/Cooking Themed**: Sous Chef Sam, Spice Master, The Prep Pro, etc.
- **Punny Food Names**: Avocado Toast, Cheese Please, Egg-cellent Chef, etc.
- **Kitchen Personalities**: The Perfectionist, The Improviser, The Speed Demon, etc.
